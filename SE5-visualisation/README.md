# SE5 — Visualisation & Dashboard

## Objectif
Dashboard interactif affichant les KPIs du pipeline Big Data en temps réel.

## Outil choisi
**Dash (Plotly)** — Python, open-source, 100% intégrable avec notre stack

## Installation
```bash
pip install dash plotly pandas
```

## Lancement
```bash
cd SE5-visualisation
python src/app.py
```
Ouvre ensuite : http://localhost:8050

## 🚀 Description du projet

Ce projet s'inscrit dans le cadre du module **Big Data** de la filière SNIA S2 à la Faculté des Sciences de Rabat. L'objectif est de concevoir et déployer un pipeline Big Data complet, de l'ingestion des données brutes jusqu'à leur visualisation en temps réel.

### Objectifs principaux

- Mettre en place un **mini-cluster distribué** (SE1)
- Implémenter un pipeline d'**ingestion de données** à grande échelle (SE2)
- Réaliser le **traitement et la transformation ETL** des données (SE3)
- Développer une couche d'**analyse en temps réel** (SE4)
- Créer un **dashboard de visualisation** interactif (SE5)
